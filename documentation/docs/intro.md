# Introduction

Welcome to the **NalamDesk** documentation!

NalamDesk is a modern, enterprise-grade **Medical Practice Management System** designed to streamline clinic operations.

## Key Features

- **Patient Management**: Efficiently register and manage patient records.
- **Queue Management**: Automated patient queuing and flow tracking.
- **Consultation**: Comprehensive consultation module with prescription generation.
- **Data Security**: Secure local-first architecture with robust encryption.
